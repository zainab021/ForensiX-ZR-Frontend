# ForensiX ZR Frontend

**ForensiX ZR Unit** is a secure crime analysis frontend system designed for police officers, administrators, and public users.  
The project provides two separate interfaces: an authorized dashboard for officials and a public interface for citizens.

## 🔐 Project Purpose

The purpose of this project is to provide a professional frontend for managing and viewing crime-related information in a structured and secure way.

## 👥 User Interfaces

### 1. Authorized Interface
For police officers/admin users.

Main modules:
- Dashboard
- Reports
- Cases
- Case Details
- Add Case
- Officers
- Users
- Analytics
- Evidence
- Notifications
- Activity Log
- Profile
- Role Management
- Settings

### 2. Public Interface
For citizens/public users.

Main modules:
- Public Dashboard
- Report Concern
- Find Help
- Safety Tips
- Emergency SOS
- My Reports
- Alerts
- Community Safety
- Contact Us
- Profile
- Settings

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Chart.js
